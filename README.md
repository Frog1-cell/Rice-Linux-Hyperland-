# Rice Linux Hyprland (∞)

**Rice Linux Hyprland (∞)** — это кастомный rice, вдохновлённый эстетикой гача-игр, таких как **Arknights: Endfield**, **Zenless Zone Zero** и **Honkai: Star Rail**.  
Вся система — от цветовой палитры до обоев — оформлена специально для фанатов этих вселенных.

- Rofi

## Важная информация

Данная конфигурация создана под мою систему.  

Если вы хотите использовать её у себя, потребуется:
- вручную обновить системные параметры  
- изменить пути к файлам  
- адаптировать конфиги под своё окружение  

**Примечание:**  
Этот rice ориентирован в первую очередь на визуальный стиль и атмосферу, а не универсальность.  
Поэтому может потребоваться дополнительная настройка под вашу систему.
