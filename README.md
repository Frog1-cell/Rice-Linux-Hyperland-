Rice Linux Hyprland (∞)

Rice Linux Hyprland (∞) — это кастомный rice, вдохновлённый эстетикой игры Arknights: Endfield.
Вся система — от цветовой палитры до обоев — оформлена специально для фанатов этой вселенной.

Особенности:
    Дизайн, вдохновлённый Arknights: Endfield
    Продуманная цветовая палитра
    Кастомные обои и визуальные элементы

!!!Важно!!!
    Данная конфигурация создана под мою систему.

Если вы хотите использовать её у себя, вам потребуется:
    вручную обновить системные параметры
    изменить пути к файлам
    адаптировать конфиги под своё окружение
